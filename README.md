# MoonWalker

Manage and strategic game on moon exploration

## Requirements

- nodejs >= 6.9.5
- yarn >= 0.27.5

## Installation

    git clone git@git.nbcorp.fr:nightbringer/moonwalker.git
    cd moonwalker
    yarn install

## Build

    yarn run server:dev

Dev server at: <http://localhost:9000/>
